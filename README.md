# OBD-C Y-splitter
Arrangement - Female-Male-Female

- compact size - 1x2.5cm
- possibility to add 120ohm CAN termination on CAN1 and CAN3 (note they can flip, so always terminate both)
- test points for sbu1 and sbu2
